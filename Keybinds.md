# Keyboard Shortcuts

All shortcuts use `Cmd` on macOS (or `Ctrl` on Windows/Linux).

## Timer Controls

| Shortcut | Action | Notes |
|----------|--------|-------|
| `Cmd+Shift+.` | Start stopwatch | Switches to Stopwatch tab and starts timer. Only works when no timer is active. |
| `Cmd+Shift+L` | Start 30-minute countdown | Switches to Countdown tab and starts a 30-minute timer. Only works when no timer is active. |
| `Cmd+Shift+J` | Start 1-hour countdown | Switches to Countdown tab and starts a 1-hour timer. Only works when no timer is active. |
| `Cmd+Enter` | Multi-purpose action | Context-aware command (see below). |

## Cmd+Enter Context-Aware Actions

The `Cmd+Enter` shortcut performs different actions based on the current tab and state:

### Stopwatch Tab
- **Not started**: Starts the stopwatch
- **Running**: Finishes the stopwatch and adds time to today's total

### Countdown Tab
- **Not started**: Starts countdown with current input values (if valid)
- **Running or paused**: Finishes countdown and adds elapsed time to today's total

### Manual Tab
- **Always**: Adds time to today's total based on current input values (if valid)

## UI Controls

| Shortcut | Action | Notes |
|----------|--------|-------|
| `Cmd+Shift+U` | Toggle task list | Shows/hides the task list panel on the right side. |
